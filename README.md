# C Quiz Game

A simple terminal-based quiz game written in C. a learning project btw!

---

### Features

- 10 multiple-choice questions
- Score tracking
- Percentage calculation
- Input validation

---

### Concepts that i got hands on 

- Functions
- Function Arguments
- Conditionals
- User Input
- Return Values

---

### Future Improvements

- Arrays
- Loops
- 100+ Questions with tier handling 
- Score Saving
- Better Input Handling

### How to Compile and Run (With gcc, clang or minGW compiler installed)

```bash

**With gcc:**

gcc c-quiz.c -o quiz
./quiz

**With Clang:**

clang c-quiz.c -o quiz
./quiz

**With MinGW:**

gcc c-quiz.c -o quiz
quiz.exe

```
